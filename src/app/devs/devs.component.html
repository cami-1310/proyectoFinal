Sobre el sitio web

Este sitio fue desarrollado utilizando Angular como framework principal, haciendo uso de TypeScript para estructurar la lógica del lado del cliente, y HTML junto con CSS para construir y diseñar la interfaz de usuario. Es importante mencionar que este proyecto abarca por el momento solo la parte del frontend.
